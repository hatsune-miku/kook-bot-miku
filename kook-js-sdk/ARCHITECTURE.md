## 背景介绍和具体需求

`Miku 机器人`项目是一个用 TypeScript 实现的 KOOK 机器人，由我（KOOK 员工）完成，里面封装了一个 WebSocket 客户端实现，用于通过 WebSocket 方式实现 KOOK 机器人协议。这个实现有幸被选为即将对外开放的官方 JS Bot SDK，但是目前还存在以下问题：

- 和`Miku 机器人`项目耦合度有点高，需要单独抽出来
- 肯定是存在 Bug 的，特别是 WS 状态机那部分
- 缺乏测试覆盖
- 缺乏详细的文档（API 文档等）
- 对 KOOK 开发者平台中所属的概念和接口实现不完整，只实现了我自己用到的
- 缺乏容器化部署脚本
